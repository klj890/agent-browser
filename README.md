## Layout

```
apps/
  main/         Electron main process (Node)
  renderer/     Vite + React shell (address bar, tabs, sidebar)
packages/
  browser-tools/   ref-based snapshot, act, content boundary
build/            electron-builder resources (icons)
e2e/              acceptance tests (PLAN scenarios 1–12)
scripts/          verify:cookie-leak, verify:injection, package:dry
spike/            Stage 2 pre-flight validation
docs/             user / admin / security docs
```

## Commands

### Development

```bash
pnpm install
pnpm dev:renderer       # Vite at http://localhost:5273
pnpm dev:main           # Electron main (expects RENDERER_URL env)
pnpm test               # unit tests (vitest)
pnpm e2e                # end-to-end acceptance suite (PLAN §Verification)
pnpm check              # biome lint + format check
pnpm format             # biome format --write
```

### Verification (security gates)

```bash
pnpm verify:cookie-leak  # end-to-end cookie redaction check
pnpm verify:injection    # prompt-injection matrix (PLAN I1–I10)
pnpm verify:all          # both
pnpm spike:ref           # Stage 2 ref-stability spike (requires playwright)
```

### Packaging (Stage 8)

```bash
pnpm build:all           # compile apps/main (tsc) + apps/renderer (vite)
pnpm package:dry         # validate electron-builder.yml (no artefacts produced)
pnpm package:mac         # .dmg + .zip for macOS (arm64 + x64)
pnpm package:win         # NSIS installer + portable .exe for Windows x64
pnpm package:linux       # AppImage + .deb for Linux x64
```

Outputs land in `release/`. The icon at `build/icon.png` is a placeholder
generated by `node build/generate-icon.mjs` — replace before shipping a
real release.
